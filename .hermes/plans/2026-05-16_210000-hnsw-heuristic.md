# HNSW Heuristic Selection + Parallelism Plan

> **Goal:** Replace the current `shrink_connections` (simple truncation) with the diversity-based heuristic and parallelize distance computations in `search_layer`.

**Architecture:** The `shrink_connections` method in `src/index/hnsw.rs` currently takes the M closest neighbors to the insertion point. The heuristic adds a diversity criterion: a candidate is only kept if it's closer to the insertion point than to any already-selected neighbor. The distance computations in `search_layer` and `shrink_connections` will use `rayon::par_iter()` to compute distances in parallel.

**Tech Stack:** Rust, rayon (already in deps)

Replace with heuristic version:
```rust
fn shrink_connections(&mut self, layer: usize, node: usize, m_max: usize, centre: &[f32]) {
    if self.graphs[layer][node].len() <= m_max {
        return;
    }

    // Phase 1: score all candidates (parallelized)
    let mut candidates: Vec<(usize, f32)> = self.graphs[layer][node]
        .par_iter()
        .map(|&n| {
            let s = distance::score(centre, &self.documents[n].embedding, self.config.metric);
            (n, s)
        })
        .collect();

    // Sort by score descending (closest first)
    candidates.sort_by(|a, b| b.1.partial_cmp(&a.1).unwrap_or(Ordering::Equal));

    // Phase 2: heuristic selection
    let mut results = Vec::with_capacity(m_max);
    let mut discard: Vec<usize> = Vec::with_capacity(candidates.len());

    for &(cand_id, cand_score) in &candidates {
        if results.len() >= m_max {
            discard.push(cand_id);
            continue;
        }

        // Check diversity: cand must be closer to centre than to any existing result
        let is_diverse = results.iter().all(|&(r_id, _)| {
            let d_cand_r = distance::score(
                &self.documents[cand_id].embedding,
                &self.documents[r_id].embedding,
                self.config.metric,
            );
            // cand_score >= d_cand_r means cand is closer to centre than to r
            cand_score >= d_cand_r
        });

        if is_diverse {
            results.push((cand_id, cand_score));
        } else {
            discard.push(cand_id);
        }
    }

    // Fill remaining slots from discard (keep closest first)
    if results.len() < m_max && !discard.is_empty() {
        discard.sort_by(|a, b| {
            let sa = distance::score(centre, &self.documents[*a].embedding, self.config.metric);
            let sb = distance::score(centre, &self.documents[*b].embedding, self.config.metric);
            sb.partial_cmp(&sa).unwrap_or(Ordering::Equal)
        });
        for &d in discard.iter().take(m_max - results.len()) {
            results.push((d, 0.0));
        }
    }

    self.graphs[layer][node] = results.into_iter().map(|(n, _)| n).collect();
}
```

Current code in `search_layer`:

```rust
for &nei in &self.graphs[layer][current.node] {
    if visited[nei] {
        continue;
    }
```

Replace with parallel-friendly version:
```rust
// Phase 1: compute distances in parallel, collect (nei, nei_score, first_time)
let neighbors: Vec<(usize, f32)> = self.graphs[layer][current.node]
    .par_iter()
    .filter(|&&nei| !visited[nei])
    .map(|&nei| {
        let s = distance::score(query, &self.documents[nei].embedding, self.config.metric);
        (nei, s)
    })
    .collect();

// Phase 2: mutate state sequentially
for (nei, nei_score) in neighbors {
    visited[nei] = true;

    candidates.push(Candidate {
        score: nei_score,
        node: nei,
    });

    if results.len() < ef {
        results.push(Reverse(Candidate {
            score: nei_score,
            node: nei,
        }));
    } else if let Some(Reverse(worst)) = results.peek() {
        if nei_score > worst.score {
            results.pop();
            results.push(Reverse(Candidate {
                score: nei_score,
                node: nei,
            }));
        }
```

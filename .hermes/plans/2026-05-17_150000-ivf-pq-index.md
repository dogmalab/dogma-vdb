# IVF-PQ Index — Implementation Plan

> **Goal:** Add a minimal IVF+PQ (Inverted File + Product Quantization) index backend to dogma-vdb.

**Architecture:** IVF clusters vectors into K groups via simple K-Means. PQ compresses each vector into M bytes by splitting it into M subvectors and quantizing each with a 256-centroid codebook. Search uses asymmetric distance computation (precomputed lookup tables). Roughly 4-64× compression depending on dimensions.

## Design Overview

### IVFPqConfig

```rust
pub struct IvfPqConfig {
    pub n_clusters: usize,    // K — number of IVF clusters (e.g. sqrt(N), default 256)
    pub n_subvectors: usize,  // M — number of PQ subvectors (e.g. D/16, default 8)
    pub n_probe: usize,       // clusters to probe per search (default 8)
    pub metric: Metric,
}
```

### IVFPqIndex

```
documents: Vec<Document>
config: IvfPqConfig

// IVF
centroids: Vec<Vec<f32>>     // [K][D] — cluster centroids
clusters: Vec<Vec<usize>>    // [K] — doc indices per cluster

// PQ
codebooks: Vec<Vec<Vec<f32>>> // [M][256][subdim] — one codebook per subvector
codes: Vec<Vec<u8>>           // [N][M] — M bytes per document
```

### K-Means helper

Simple function: random init, fixed iterations, nearest-assign, mean-update.

```rust
fn kmeans(data: &[Vec<f32>], k: usize, max_iter: usize, metric: Metric) -> Vec<Vec<f32>>
```

### Search algorithm

```
1. Split query into M subvectors (each subdim = D/M)
2. Compute query-to-centroid distance for all K centroids → select top n_probe
3. Precompute lookup tables: for each subvector m, distance(query_sub[m], codebook[m][c]) for c in 0..255
4. For each doc in selected clusters:
     score = sum(lut[m][codes[doc][m]] for m in 0..M)
5. Global sort, return top-k
```

### Asymmetric distance

"asymmetric" porque query viene en f32 y los códigos están en u8. Precomputamos la distancia del query a cada centroide del codebook, y para cada doc solo sumamos M valores de la tabla. Esto evita computar distancias f32 por documento.

```rust
use crate::distance::{self, Metric};
use crate::doc::Document;
use crate::index::{self, Index, ScoredDocument};
use rayon::prelude::*;
use std::cmp::Ordering;

/// IVF-PQ configuration.
#[derive(Debug, Clone)]
pub struct IvfPqConfig {
    /// Number of IVF clusters (K). Default: 256.
    pub n_clusters: usize,
    /// Number of PQ subvectors (M). D must be divisible by M. Default: 8.
    pub n_subvectors: usize,
    /// Number of clusters to probe per search. Default: 8.
    pub n_probe: usize,
    /// Distance metric.
    pub metric: Metric,
}

impl Default for IvfPqConfig {
    fn default() -> Self {
        Self {
            n_clusters: 256,
            n_subvectors: 8,
            n_probe: 8,
            metric: Metric::Cosine,
        }
    }
}

/// Simple K-Means clustering.
///
/// Returns K centroids (each D-dimensional). Uses random initialization
/// (pick K random data points), fixed iterations, convergence check via
/// centroid movement threshold. Pure Rust, no external dep.
fn kmeans(data: &[Vec<f32>], k: usize, max_iter: usize, metric: Metric) -> Vec<Vec<f32>> {
    if data.is_empty() || k == 0 {
        return Vec::new();
    }
    let dim = data[0].len();
    let n = data.len();

    // 1. Initialize: pick k random distinct data points as centroids
    let mut rng = 42u64;
    let mut centroids: Vec<Vec<f32>> = Vec::with_capacity(k);
    let mut used = vec![false; n];
    for _ in 0..k.min(n) {
        // Simple deterministic "random" selection
        rng = rng.wrapping_mul(6364136223846793005).wrapping_add(1442695040888963407);
        let idx = (rng >> 33) as usize % n;
        if !used[idx] {
            centroids.push(data[idx].clone());
            used[idx] = true;
        } else {
            // Find next unused
            for i in 0..n {
                if !used[i] {
                    centroids.push(data[i].clone());
                    used[i] = true;
                    break;
                }
            }
        }
    }

    // Fill remaining centroids if k > n (unlikely but safe)
    while centroids.len() < k {
        centroids.push(vec![0.0; dim]);
    }

    // 2. Iterate
    let mut assignments = vec![0usize; n];
    for _iter in 0..max_iter {
        // Assignment step
        let mut changed = false;
        for (i, vec) in data.iter().enumerate() {
            let mut best = 0;
            let mut best_score = f32::NEG_INFINITY;
            for (j, c) in centroids.iter().enumerate() {
                let s = distance::score(vec, c, metric);
                if s > best_score {
                    best_score = s;
                    best = j;
                }
            }
            if assignments[i] != best {
                assignments[i] = best;
                changed = true;
            }
        }
        if !changed {
            break; // Converged
        }

        // Update step
        let mut sums: Vec<Vec<f32>> = vec![vec![0.0f32; dim]; k];
        let mut counts = vec![0usize; k];
        for (i, vec) in data.iter().enumerate() {
            let c = assignments[i];
            for (d, &v) in vec.iter().enumerate() {
                sums[c][d] += v;
            }
            counts[c] += 1;
        }
        for (j, sum) in sums.iter().enumerate() {
            if counts[j] > 0 {
                let inv = 1.0 / counts[j] as f32;
                for d in 0..dim {
                    centroids[j][d] = sum[d] * inv;
                }
            }
        }
    }

    centroids
}
```

## Task 2: IvfPqIndex — struct + new() + insert()

**File:** `src/index/ivf_pq.rs` (lines ~80-250)

```rust
pub struct IvfPqIndex {
    documents: Vec<Document>,
    config: IvfPqConfig,

    // IVF
    centroids: Vec<Vec<f32>>,
    clusters: Vec<Vec<usize>>,

    // PQ
    codebooks: Vec<Vec<Vec<f32>>>,  // [M][256][subdim]
    codes: Vec<Vec<u8>>,            // [N][M]
}

impl IvfPqIndex {
    pub fn new(config: IvfPqConfig) -> Self {
        Self {
            documents: Vec::new(),
            config,
            centroids: Vec::new(),
            clusters: Vec::new(),
            codebooks: Vec::new(),
            codes: Vec::new(),
        }
    }

    pub fn len(&self) -> usize { self.documents.len() }
    pub fn is_empty(&self) -> bool { self.documents.is_empty() }
    pub fn documents(&self) -> &[Document] { &self.documents }
    pub fn config(&self) -> &IvfPqConfig { &self.config }

    /// Insert documents and train IVF+PQ from scratch.
    pub fn insert(&mut self, docs: &[Document]) {
        // Filter out docs without embeddings
        let valid: Vec<Document> = docs.iter().filter(|d| !d.embedding.is_empty()).cloned().collect();
        if valid.is_empty() {
            return;
        }

        // Append to existing
        let offset = self.documents.len();
        self.documents.extend(valid);

        // Rebuild entire index from all docs
        self.build_index();
    }

    /// Train IVF + PQ from the full document set.
    fn build_index(&mut self) {
        let docs = &self.documents;
        let n = docs.len();
        if n == 0 {
            return;
        }

        let dim = docs[0].embedding.len();
        let m = self.config.n_subvectors;
        let subdim = dim / m;

        // Collect all embeddings as Vec<Vec<f32>>
        let all_vecs: Vec<Vec<f32>> = docs.iter().map(|d| d.embedding.clone()).collect();

        // ---- IVF: K-Means on full vectors ----
        let k = self.config.n_clusters.min(n);
        self.centroids = kmeans(&all_vecs, k, 20, self.config.metric);

        // Assign each doc to nearest centroid
        self.clusters = vec![Vec::new(); k];
        for (i, vec) in all_vecs.iter().enumerate() {
            let mut best = 0;
            let mut best_score = f32::NEG_INFINITY;
            for (j, c) in self.centroids.iter().enumerate() {
                let s = distance::score(vec, c, self.config.metric);
                if s > best_score {
                    best_score = s;
                    best = j;
                }
            }
            self.clusters[best].push(i);
        }

        // ---- PQ: train codebooks ----
        // For each of M subvector spaces, run K-Means with 256 centroids
        let codebook_size = 256usize;
        self.codebooks = Vec::with_capacity(m);

        for sub_idx in 0..m {
            // Collect all subvectors for this sub-space
            let start = sub_idx * subdim;
            let end = start + subdim;
            let subvecs: Vec<Vec<f32>> = all_vecs
                .iter()
                .map(|v| v[start..end].to_vec())
                .collect();

            // Train 256 centroids for this sub-space
            let cb = kmeans(&subvecs, codebook_size, 20, Metric::Euclidean);
            self.codebooks.push(cb);
        }

        // ---- PQ: encode all documents ----
        self.codes = Vec::with_capacity(n);
        for vec in &all_vecs {
            let mut code = Vec::with_capacity(m);
            for sub_idx in 0..m {
                let start = sub_idx * subdim;
                let end = start + subdim;
                let subvec = &vec[start..end];

                // Find nearest centroid in this sub-space
                let mut best = 0u8;
                let mut best_dist = f32::NEG_INFINITY;
                // Use Euclidean distance for PQ code assignment
                for (c_idx, centroid) in self.codebooks[sub_idx].iter().enumerate() {
                    let s = distance::score(subvec, centroid, Metric::Euclidean);
                    if s > best_dist {
                        best_dist = s;
                        best = c_idx as u8;
                    }
                }
                code.push(best);
            }
            self.codes.push(code);
        }
    }
}
```

## Task 3: IvfPqIndex — search() + Index trait impl

**File:** `src/index/ivf_pq.rs` (lines ~250-350)

```rust
    /// Search using IVF-PQ: select top n_probe clusters, then
    /// asymmetric distance computation via precomputed lookup tables.
    pub fn search(&self, query: &[f32], k: usize) -> Vec<ScoredDocument> {
        if self.documents.is_empty() || k == 0 {
            return Vec::new();
        }

        let m = self.config.n_subvectors;
        let subdim = query.len() / m;
        let probe = self.config.n_probe.min(self.centroids.len());

        // 1. Find top n_probe clusters by query-to-centroid distance
        let mut centroid_scores: Vec<(usize, f32)> = self
            .centroids
            .par_iter()
            .enumerate()
            .map(|(i, c)| {
                let s = distance::score(query, c, self.config.metric);
                (i, s)
            })
            .collect();
        centroid_scores.sort_unstable_by(|a, b| b.1.partial_cmp(&a.1).unwrap_or(Ordering::Equal));
        let active_clusters: Vec<usize> = centroid_scores
            .into_iter()
            .take(probe)
            .map(|(i, _)| i)
            .collect();

        // 2. Precompute PQ lookup tables: [M][256] — asymmetric distance
        //    from query subvectors to codebook centroids.
        let mut luts: Vec<Vec<f32>> = Vec::with_capacity(m);
        for sub_idx in 0..m {
            let start = sub_idx * subdim;
            let end = start + subdim;
            let q_sub = &query[start..end];
            let cb = &self.codebooks[sub_idx];
            let mut lut = Vec::with_capacity(256);
            for c in cb {
                let s = distance::score(q_sub, c, self.config.metric);
                lut.push(s);
            }
            luts.push(lut);
        }

        // 3. Scan docs in selected clusters using lookup tables
        let mut results: Vec<ScoredDocument> = active_clusters
            .par_iter()
            .flat_map(|&ci| {
                let cluster = &self.clusters[ci];
                let mut local: Vec<ScoredDocument> = cluster
                    .iter()
                    .map(|&doc_id| {
                        let code = &self.codes[doc_id];
                        let mut score = 0.0f32;
                        for (sub_idx, &c) in code.iter().enumerate() {
                            score += luts[sub_idx][c as usize];
                        }
                        ScoredDocument {
                            score,
                            document: self.documents[doc_id].clone(),
                        }
                    })
                    .collect();
                // Sort each cluster locally
                local.sort_unstable_by(|a, b| {
                    b.score.partial_cmp(&a.score).unwrap_or(Ordering::Equal)
                });
                local
            })
            .collect();

        // 4. Global sort and truncate
        results.sort_unstable_by(|a, b| {
            b.score.partial_cmp(&a.score).unwrap_or(Ordering::Equal)
        });
        results.truncate(k);
        results
    }

impl Index for IvfPqIndex {
    fn insert(&mut self, docs: &[Document]) { self.insert(docs); }
    fn documents(&self) -> &[Document] { &self.documents }
    fn len(&self) -> usize { self.documents.len() }

    fn delete(&mut self, ids: &[&str]) -> usize {
        let before = self.documents.len();
        let id_set: std::collections::HashSet<&str> = ids.iter().copied().collect();
        let remaining: Vec<Document> = self.documents.iter().filter(|d| !id_set.contains(d.id.as_str())).cloned().collect();
        let deleted = before - remaining.len();
        if deleted > 0 {
            let config = self.config.clone();
            *self = Self::new(config);
            self.documents = remaining;
            self.build_index();
        }
        deleted
    }

    fn search(&self, query: &[f32], k: usize) -> Vec<ScoredDocument> {
        self.search(query, k)
    }

    fn search_filtered(&self, query: &[f32], k: usize, filter: &(dyn Fn(&Document) -> bool + Sync)) -> Vec<ScoredDocument> {
        // Post-filter: search with multiplier, then filter
        let multiplier = (k * 3).max(self.config.n_probe * 2);
        let results = self.search(query, multiplier);
        results.into_iter().filter(|r| filter(&r.document)).take(k).collect()
    }
}
```

```rust
#[cfg(test)]
mod tests {
    use super::*;

    fn make_doc(id: &str, embedding: Vec<f32>) -> Document {
        Document::builder(id, id).embedding(embedding).build()
    }

    fn small_config() -> IvfPqConfig {
        IvfPqConfig {
            n_clusters: 4,
            n_subvectors: 2,
            n_probe: 2,
            metric: Metric::Cosine,
        }
    }

    #[test]
    fn test_empty_index() {
        let idx = IvfPqIndex::new(small_config());
        assert!(idx.search(&[1.0, 0.0], 5).is_empty());
        assert!(idx.is_empty());
    }

    #[test]
    fn test_single_insert() {
        let mut idx = IvfPqIndex::new(small_config());
        idx.insert(&[make_doc("a", vec![1.0, 0.0, 0.0, 0.0])]);
        assert_eq!(idx.len(), 1);
        let results = idx.search(&[1.0, 0.0, 0.0, 0.0], 5);
        assert_eq!(results.len(), 1);
        assert_eq!(results[0].document.id, "a");
    }

    #[test]
    fn test_search_returns_closest() {
        let mut idx = IvfPqIndex::new(small_config());
        idx.insert(&[
            make_doc("a", vec![1.0, 0.0, 0.0, 0.0]),
            make_doc("b", vec![0.0, 1.0, 0.0, 0.0]),
        ]);
        let results = idx.search(&[1.0, 0.0, 0.0, 0.0], 2);
        assert_eq!(results.len(), 2);
        assert_eq!(results[0].document.id, "a");
    }

    #[test]
    fn test_recall_against_bf() {
        // Small test: 50 docs, compare with BF
        let mut bf = BruteForceIndex::new(Metric::Cosine);
        let mut ivf = IvfPqIndex::new(IvfPqConfig {
            n_clusters: 5,
            n_subvectors: 2,
            n_probe: 3,
            metric: Metric::Cosine,
        });

        let mut docs = Vec::with_capacity(50);
        for i in 0..50 {
            let angle = i as f64 * 0.12566;
            docs.push(make_doc(
                &format!("d{}", i),
                vec![angle.cos() as f32, angle.sin() as f32, 0.0, 0.0],
            ));
        }
        bf.insert(&docs);
        ivf.insert(&docs);

        let query = vec![1.0, 0.0, 0.0, 0.0];
        let bf_res = bf.search(&query, 5);
        let ivf_res = ivf.search(&query, 5);

        // At least 2 of top-5 should match (40%) — PQ is approximate
        let bf_ids: std::collections::HashSet<&str> =
            bf_res.iter().map(|r| r.document.id.as_str()).collect();
        let hits = ivf_res.iter().filter(|r| bf_ids.contains(r.document.id.as_str())).count();
        assert!(hits >= 2, "IVF-PQ recall too low: {}/5", hits);
    }
```

## Constraints & Notes

- D must be divisible by M (n_subvectors). Assert or return error if not.
- K-Means is intentionally simple: random init, Euclidean metric for PQ subvectors, cosine for IVF.
- PQ codebooks use Euclidean distance for training (standard practice — subvectors aren't normalized).
- PQ encoding also uses Euclidean distance (nearest centroid in subvector space).
- Search uses the configured metric for the lookup table (cosine ≈ dot).
- No SIMD optimization for now — the lookup table approach is already fast.
- No `unsafe`. No new dependencies.
